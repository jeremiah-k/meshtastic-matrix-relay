# MMRelay 1.3 migration guide (archived)

This guide is no longer part of the current MMRelay setup path.

Systems still running MMRelay 1.2.x or older should install a 1.3.x release,
follow the [archived 1.3 migration guide](dev/archive/MIGRATION_1.3.md), and then
upgrade to MMRelay 1.4.0 or newer.
